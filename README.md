# Passive Filter Design and Frequency Response Analysis using LTspice ⚡

This project presents the design and simulation of passive filters using LTspice.

The project includes low-pass, high-pass, and band-pass filter implementations and investigates their frequency response characteristics.

Full simulation models are provided to allow complete reproduction.

## 🎯 Project Objectives

This project investigates the frequency response of passive filters.

The experiment includes:

Low-pass filter design  
High-pass filter design  
Band-pass filter design  

The cutoff frequency is verified using simulation and theoretical analysis.

For example, the low-pass filter cutoff frequency was calculated as:

15915.5 Hz

and simulation result:

15.86 kHz
